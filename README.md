# Mobile Comparison Website
This website allows users to compare different mobile phones based on their specifications. Users can select the features they are interested in and the website will generate a list of phones that meet their criteria.

# Features
Compare mobile phones based on their specifications, such as price, screen size, camera quality, and battery life.
Save your favorite phones and compare them later.
Share your comparisons with others.
Installation
To install the website, you will need to have the following:

A web server, such as Apache or Nginx.
A database, such as MySQL or PostgreSQL.
PHP 7 or higher.
Once you have installed the necessary software, you can download the website from GitHub.

# Usage
To use the website, open a web browser and navigate to the website's URL. You will be greeted with a homepage that lists all of the available phones.

To compare two phones, click on the "Compare" button next to each phone. The website will then generate a comparison table that shows the specifications of the two phones.

You can also save your favorite phones by clicking on the "Save" button next to each phone. Your saved phones will be displayed in the "Saved Phones" section of the website.

To share your comparisons with others, click on the "Share" button next to each comparison. The website will then generate a shareable link that you can send to others.
